# github-split

### install with curl

```bash
curl raw.github.com/jeromedecoster/github-split/master/script.sh \
    --location \
    --silent \
    | bash
```

### install with wget

```bash
wget raw.github.com/jeromedecoster/github-split/master/script.sh \
    --output-document=- \
    --quiet \
    | bash
```
